How to edit home page video background

Lines 574 custom.js

In corporate version we used a full screen video background for home page. there is no extra html markup and it has been handled from javascript.

First let's see the folder structure inside "assets" folder you will find a "video" folder. create a new folder inside "video" folder and give it a name let's say "myvideo" then add your video file to this folder. You need 3 format of your video for run in different browsers("mp4","ogv" and "webm") and give all of them a same name like myvideo. Additionally you need to add a single image with the same name for callback if video failed

Ok now we have our files let's get to javascript. Open "custom.js" file inside "assets/js" and arround line 574 you should see this:

/* home video background
----------------------------------------------*/
var vs, videoBackground = {
  settings : {
    videoName : 'goldpai/goldpai',
  },

Just change the video name to "myvideo/myvideo". first one is folder name and second one is file name.
