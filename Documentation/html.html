Template main layout

This template is using bootstrap 3 css framework so you might wanna take a look at their documentation which is available at Bootstrap documentation

First lets see how the whole thing stands in term of HTML markup.

We have a wrapper which is acting like a dummy body and contains the whole template. This template is one page so all contents should be in one html file which is corporate.html or personal.html.

#upstream is a higher level which we used for fullscreen gallery and portfolio and also blog posts.

#content is the wrapper of all pages

#menu-panel is our navigation menu

Each menu item is a li inside the ul with slider-containerid and has a title, subtitle and an image. menu title should be entered inside h2 tag and the subtitle in h3 tag. That 01 number is optional but recommanded for better user experience. finally each menu needs an image which is a simple img tag. To add menu item just copy and paste one of these li and edit it's information.

Notice: The order of menu items should be same as order of pages inside swiper. so the first menu refer to first page and second menu to second page and so on.
