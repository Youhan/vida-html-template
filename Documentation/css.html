Change the skin

We have 3 pre defined skin. To change the default skin all you need to edit html file.in both corporate.html and personal.html at line 16:

Each one of styles has a css file located in "assets/css/skins". currently you have three option either use skin2 or skin3 in above line or just delete it and you will have the default blue skin.

Supppose we want to change to red.css we need to open red.css file and pick the color which is #d02b2b in this case and enter it in barColor



Add new skin

You can have your own colors these 3 predefined are just some examples and we will als add some more in next versions. to add a new one just copy and paste "skin3" folder and let's name it "skin4" . here we have a main color which is #272822 for our elements you can change all occurence of it with your own color but then you need to sync the other colors too

Notice:

Team carousel section has different color for different items but all colors all from a same range. We used javascript to generate some colors in a same range based on an input color. that main color is at line 178 of "style.css" in skins folder. here is from skin3 

  .team-container .team-carousel-wrapper,
  .team-container .team-carousel .item{
    background: #272822;
  }

You need to set this color based on your main color.

Notice:

There is one image which is belong to menu trigger button you need to give it your main color we included the PSD file of this image so you can easily change the color.
